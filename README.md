# MASK URL
A simple url masking tool

<h3 align="left">INSTALL :</h3>

````bash
pkg update && upgrade -y
pkg install python -y
git clone https://github.com/ShTasrif/MaskURL
````

## How to use 

#### Run :

````bash
cd MaskURL
python mask.py
````
